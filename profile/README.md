**Installation Guide**  
1. Download the latest archive.  
2. Extract to a dedicated folder.  
3. Run `StarCitizenTrainer.exe` as administrator.  
4. Launch Star Citizen (wait for full load into PU).  
5. Press INSERT to open the trainer interface.  
6. Activate desired features via hotkeys or menu.  

Re-attach after game restarts, server jumps, or major patches.
